**Plain Craft Launcher 2 Community Edition - C++ Reimplementation**

PCL2-CE-CPP is a modern, high-performance Minecraft launcher core and UI, reimagined in **C++20** and **WinUI 3 (Windows App SDK)**. It serves as a C++ port of the [PCL2 Community Edition (PCL2-CE)](https://github.com/PCL-Community/PCL2-CE) (and also [Plain Craft Launcher 2 (PCL2)](https://github.com/Meloong-Git/PCL)).
